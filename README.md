# AI FAQ Chatbot

This project is developed for the CodeAlpha Artificial Intelligence Internship.

## Features

- NLP-based FAQ matching
- TF-IDF Vectorization
- Cosine Similarity
- Interactive Streamlit UI

## Technologies

- Python
- Streamlit
- NLTK
- Scikit-Learn

## Run

pip install -r requirements.txt

streamlit run app.py
